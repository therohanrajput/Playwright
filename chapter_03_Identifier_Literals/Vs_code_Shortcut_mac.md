# VS Code Shortcuts for macOS

## Navigation
- `Cmd + P` : Quick open file by name
- `Cmd + Shift + O` : Go to symbol in file
- `Cmd + Shift + E` : Open Explorer
- `Cmd + Shift + F` : Search across files
- `Cmd + Shift + G` : Open Source Control
- `Cmd + B` : Toggle sidebar visibility

## Editing
- `Cmd + /` : Toggle line comment
- `Option + Shift + A` : Toggle block comment
- `Option + Up/Down` : Move line or selection up/down
- `Shift + Option + Right/Left` : Expand/shrink selection
- `Cmd + D` : Add next occurrence to selection
- `Cmd + L` : Select entire line
- `Cmd + Shift + K` : Delete line
- `Cmd + Enter` : Insert line below
- `Cmd + Shift + Enter` : Insert line above
- `Cmd + X` : Cut line (when no selection)

## Multi-cursor
- `Option + Click` : Insert cursor at multiple points
- `Cmd + Option + Down/Up` : Add cursor below/above
- `Cmd + Shift + L` : Select all occurrences of current selection

## Search and Replace
- `Cmd + F` : Find in file
- `Cmd + Option + F` : Replace in file
- `Cmd + Shift + F` : Find in workspace
- `Cmd + Shift + H` : Replace in workspace

## File and Editor Management
- `Cmd + N` : New file
- `Cmd + S` : Save file
- `Cmd + Option + S` : Save all files
- `Cmd + W` : Close editor tab
- `Cmd + Shift + T` : Reopen closed editor
- `Cmd + \` : Split editor
- `Cmd + 1`, `Cmd + 2`, `Cmd + 3` : Focus first, second, third editor group

## Debugging
- `F5` : Start/continue debugging
- `Shift + F5` : Stop debugging
- `F9` : Toggle breakpoint
- `F10` : Step over
- `F11` : Step into
- `Shift + F11` : Step out
- `Cmd + Shift + Y` : Toggle Debug Console

## Integrated Terminal
- `` Ctrl + ` `` : Toggle terminal
- `Cmd + Shift + C` : Open new terminal
- `Cmd + K` then `` Ctrl + ` `` : Clear terminal

## Git and Source Control
- `Cmd + Shift + G` : Open Source Control view
- `Cmd + Enter` (inside commit message box) : Commit
- `Ctrl + Shift + G` : Stage selected changes

## General Productivity
- `Cmd + Shift + P` : Open Command Palette
- `Cmd + K Cmd + S` : Open Keyboard Shortcuts
- `Cmd + K Z` : Toggle Zen Mode
- `Cmd + K V` : Open Markdown preview
- `Cmd + Shift + V` : Open Markdown preview to the side
