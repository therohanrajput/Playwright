# Playwright Learning Repository

This repository contains JavaScript examples and practice files for learning Playwright and JavaScript fundamentals.

## Folder structure

### Chapter 01: Basics
`chapter_01_Basics/` - Basic JavaScript setup and beginner examples
- 01_Basics.js - Initial setup
- 02_Js.js - JavaScript fundamentals
- 03_Js_Verify_Setup.js - Verification of environment setup
- 04_Hot_Code.js - Hot reload examples

### Chapter 02: JavaScript Concepts
`chapter_02_Javascript_Concepts/` - JavaScript concepts and identifier naming examples
- 05_Js_Basics.js - Core JavaScript basics

### Chapter 03: Identifier & Literals
`chapter_03_Identifier_Literals/` - Identifier/literal-related examples and supporting notes
- 06_identifier.js - Identifier basics
- 07_identifier_rules.js - Rules for identifiers
- 08_comments.js - Comment syntax and usage
- Vs_code_Shortcut_mac.md - Useful VS Code shortcuts for macOS

### Chapter 04: JavaScript Concepts (Advanced)
`chapter_04_JS_Concepts/` - In-depth JavaScript variable scoping, hoisting, and function concepts
- 09_var_let_const.js - Overview of var, let, const
- 10_function.js - Function declarations and expressions
- 11_var_explained.js - var keyword deep dive
- 12_let.js - let keyword and block scoping
- 13_const.js - const keyword and immutability
- 14_var_functionscope.js - var function scope behavior with comprehensive diff between var, let, const
- 15_let_scope.js - let block scope examples
- 16_hoisting.js - Hoisting basics
- 17_hoisting_fn.js - Function hoisting
- 18_let_hoisting.js - let hoisting and Temporal Dead Zone
- 19_let_hoisting_block.js - let hoisting in block scope
- 20_let_const.js - let and const comparison
- 21_Jr_QA.js - QA questions and answers

### Chapter 05: Literals
`chapter_05_Literal/` - Different types of literals in JavaScript
- 22_Literal.js - Literal types overview
- 23_null_undefined.js - Difference between null and undefined
- 24_null.js - null keyword deep dive
- 25_Literal.js - Literal value examples
- 26_Literal_number_all.js - All number types (integer, float, exponential, hex, octal, binary, BigInt, special values)
- 27_string.js - String literals
- 28_Template_Literals.js - Template literals with examples
- 29_Backtick_single_double.js - Comparison of backticks, single and double quotes

### Chapter 06: Operators
`chapter_06_Operator/` - JavaScript operators and operations

### Chapter 11: Arrays
`chapter_11_Arrays/` - Array methods and operations

## Notes

- Files are organized by chapter and topic.
- Examples use plain JavaScript and focus on syntax, scoping, hoisting, and practical demonstrations.
- `Vs_code_Shortcut_mac.md` contains useful VS Code shortcuts for macOS.
- Chapter 04 includes detailed explanations of var/let/const differences including scope, re-declaration, re-assignment, hoisting, initialization requirements, and global object properties.
